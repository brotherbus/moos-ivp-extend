MOOS-IvP Web Interface

This web interface is intended to provide scope MOOS variables, camera feed and other interesting stuff from the mission.

  • The Variables tab shows a table providing information such as positon, heading, battery level etc.
  • The Camera tab shows a feed of the camera from the boat. It uses multipart jpeg stream (from motion program).
  • The Info tab displays information about the project, such as authors and website links.
